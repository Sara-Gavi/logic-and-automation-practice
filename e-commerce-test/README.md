# E-commerce Test - Simulación con Promesas / Promises Simulation

Este ejercicio simula parte del backend de un e-commerce usando **JavaScript**.  
Está diseñado para practicar habilidades como **desarrolladora**.

This exercise simulates part of an e-commerce backend using **JavaScript**.  
It is designed to practice fundamental skills as a **developer**.

---

## Descripción / Description

**ES:**  
El objetivo es trabajar con promesas, manejo de errores, lógica, arrays y objetos.  
Uso APIs _simuladas_ (mock APIs) que imitan llamadas reales a un servidor.

**EN:**  
The goal is to work with promises, error handling, logic, arrays, and objects.  
I use _simulated APIs_ (mock APIs) that imitate real server calls.
